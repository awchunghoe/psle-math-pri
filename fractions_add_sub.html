Add & Sub Fraction of a Set
What it is

P4 students learn to add and subtract fractions in three situations: (1) same denominator, (2) one denominator is a multiple of the other (e.g., 2 and 4), and (3) mixed numbers. The core rule never changes: you can only add/subtract when the denominators are the same. Denominators ≤ 12 per MOE syllabus.

Case 1: Same Denominator

Rule: Add or subtract the numerators. Keep the denominator.

Example: 2/7 + 3/7 = (2+3)/7 = 5/7
Example: 5/6 - 1/6 = (5-1)/6 = 4/6 = 2/3 (simplify!)

⚠️ Common trap

Students add both numerators AND denominators: 2/7 + 3/7 = 5/14. Wrong! The denominator stays 7 — you're adding pieces of the same-sized whole.

Case 2: Different Denominators (one is a multiple)

Rule: Find the equivalent fraction so both share the same denominator. Usually the bigger denominator.

1

Identify which denominator is the multiple. For 1/2 + 1/4: 4 is a multiple of 2.

2

Convert: 1/2 = 2/4 (multiply top and bottom by 2).

3

Now add: 2/4 + 1/4 = 3/4

Another: 3/4 - 1/8 → 3/4 = 6/8 → 6/8 - 1/8 = 5/8

Case 3: Mixed Numbers

Rule: Add/subtract the whole numbers separately, then the fractions. If the fraction goes negative, borrow 1 from the whole number.

Example: 2 1/4 + 1 2/4 = (2+1) + (1/4 + 2/4) = 3 3/4
Example (borrowing): 3 1/5 - 1 3/5 → 3 1/5 = 2 6/5 → (2-1) + (6/5 - 3/5) = 1 3/5

💡 Teaching tip

Draw the mixed number as a whole number bar + a fraction bar. When borrowing, "break" one whole bar into fraction pieces. Visual learners get this instantly.

Simplification Checklist

1

Is the answer a proper fraction in lowest terms? (e.g., 4/6 → 2/3)

2

Is it an improper fraction that should be a mixed number? (e.g., 7/4 → 1 3/4)

3

Did I check if numerator = denominator? (e.g., 6/6 = 1 whole)

⬆️ P5 Extension: Once students master adding/subtracting fractions, they move to Part-Whole — using fractions as parts of a total to solve problem sums.
